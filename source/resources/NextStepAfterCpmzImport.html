1. Your ConPro project has been successfully imported. You can now continue to work on your project in Miradi. You may wish to start by reviewing the information in the summary view and then cleaning up your conceptual model diagrams and building new result chain diagrams.
  2. Miradi has created separate draft conceptual model diagrams for each conservation target plus one diagram showing “All on One Page”. Please note that there are checkboxes in the Control Bar for turning “Tags” on/off to filter the contents of the diagrams by overall threat rank.
  3. Short animated, narrated Miradi tutorials are available at: http://elearn.tnc.org/cap/miradi/player.html. Tutorials 46-50 provide guidance specific to ConPro-Miradi data exchanges.
  4. Go to http://conservationgateway.org/Miradi for more details about the import process including, in particular, how information from various ConPro data fields are handled in Miradi.
  5. If you have any problems or suggestions as to ways to improve this import process, please contact us at TNC@miradi.org.
