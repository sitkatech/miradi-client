Purple monitoring activities function in the same way as regular yellow activities, but are used to capture work specifically needed to implement project monitoring functions.

Hint: To create monitoring activities from the Work Plan view, first make sure that the table view includes Monitoring data by clicking the <Customize Table...> button and selecting either Actions and Monitoring or Monitoring from the Include Data From drop down menu. Then select the strategy in the table that will have the new monitoring activity, click the <Create...> button, and select the Create Monitoring Activity option.

Hint: To change a yellow activity to a purple monitoring activity, go to the <Lists> tab in the Strategic Plan view, and select the Activities Only table from the drop down menu. Select the activity in the table that you want to change, and click the Monitoring Activity check box in the bottom panel.

Hint: You may want to create a separate results chain to capture the status monitoring activities that you implement to track the status of the targets and threats in your Conceptual Model.
