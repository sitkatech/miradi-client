Some hints for working with your budget include:

Hint: The same resource may have different accounting codes or funding sources for different activities / methods or tasks.

Hint: The current version of Miradi only supports financial expenses that can be assigned to a specific resource.

Hint: If you need to adjust the planning horizon for your project, go to the project start and end date fields on <Planning> tab in the Summary View. You can also use the tab to adjust the currency type and the number of decimal places displayed.
