Welcome to the Threat Rating View

Miradi's default method is the Simple Threat Rating Mode (see description of the two modes in the right wizard panel). If you want to use the Stress Based Threat Rating Mode, you can select it using the drop down in the threat rating view toolbar, or the <Edit/Preferences/Threat Rating View> menu and change the Threat Rating Mode field to "Stress Based Threat Rating Mode."

The table below contains a summary of your threat ratings. Target-Threat combinations that have been rated should appear colored in. If there is a threat or target that is missing or a threat is not linked to a specific target, return to the diagram view to add the factors or links and then return to this view to rate them.
