For Key Attribute viability analysis, click on the <Summary> tab of the target properties dialog, and make sure the Viability Analysis Mode field is set to <Key Attribute>. Then click on the <Viability> tab.

The first step is to develop one or more Key Ecological Attributes (KEAs) for each target. To create a new KEA, click the <Create Key Ecological Attribute> button.  Select the new KEA in the upper panel, and then fill in the following fields in the lower panel:

  • Key Ecological Attribute: The name of the KEA.
  • Type: The type of KEA.
  • Comments: Detail about the KEA.

To edit or delete a KEA, click on the desired row in the upper panel, and then either edit the text fields in the lower panel, or click on the <Delete Key Ecological Attribute> button.
