The next step is to develop one or more indicators for each KEA.

To create a new indicator, make sure the KEA is selected in the upper panel, and click the <Create Indicator> button.  Fill in the following fields:

  • ID: A user assigned identification code for the indicator.
  • Name: The name of the indicator. Generally 2-5 words (e.g., Number of Sharks).
  • Details: Details about the indicator.
  • Comments: Discussion about the indicator.

Click on the priority and progress boxes to assign the appropriate level to each indicator.

To edit or delete an indicator, click on the indicator in the upper panel, and then either edit the fields in the lower panel, or click on the <Delete Indicator> button.
