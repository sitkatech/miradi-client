Develop Project Budgets: Accounting Codes & Funding Sources

To develop a budget, you will first want to create accounting codes and funding sources.

On the <Accounting Codes> tab, if your project already has accounting codes from your home organization, you can use the <Import Accounting Code> button to import a tab delimited text file (2 columns, code followed by description). If you have to create codes from scratch, click on the <Create Accounting Code> button to add a new one and then type the code and the label in the box at the bottom of the page. Repeat for other codes as needed.

On the <Funding Sources> tab, click on the <Create Funding Source> button to add a new one and then type the code (if needed) and the label in the box at the bottom of the page. Repeat for other sources as needed.
